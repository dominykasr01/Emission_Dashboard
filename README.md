# Global Emissions Dashboard

This project presents an interactive dashboard for analysing global greenhouse gas emissions using Python and Streamlit.

## Overview
The dashboard allows users to:
- Explore global emissions trends over time
- Filter data by year range
- Compare emissions between countries
- Identify the top emitting countries

The application uses a dataset containing emissions totals by country and year.

## Technologies Used
- Python
- Streamlit
- Pandas

## How to Run the Application

1. Clone the repository:
git clone https://github.com/dominykasr01/Emission_Dashboard.git

2. Navigate to the project folder:
cd Emission_Dashboard

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
streamlit run app.py
